# Competitor Research — Workflow/Approval Tracking Dashboards

## 1. Kissflow
- **Key Features**: Visual workflow builder, approval routing, SLA tracking, custom dashboards, process analytics
- **Approval Tracking**: Multi-level approvals, parallel/sequential routing, auto-escalation, delegation
- **Analytics**: Process bottleneck reports, cycle time analysis, SLA compliance rates
- **Strengths**: Clean UI, no-code builder, good for mid-market
- **Gaps**: Limited predictive analytics, no calendar heatmaps, basic audit trail
- **Pricing**: $1,500/mo (50 users), enterprise custom

## 2. ProcessMaker
- **Key Features**: BPMN 2.0 process designer, decision engine, API integrations, task management
- **Approval Tracking**: Multi-step approvals, conditional routing, role-based assignment
- **Analytics**: Process intelligence, dashboards, KPI tracking, bottleneck visualization
- **Strengths**: Enterprise-grade BPMN, open-source option, deep customization
- **Gaps**: Steep learning curve, UI feels dated, no real-time kanban board
- **Pricing**: $1,495/mo+ enterprise

## 3. Nintex
- **Key Features**: Workflow automation, document generation, e-signatures, RPA, process mapping
- **Approval Tracking**: Multi-level with escalation, conditional approvals, mobile approvals
- **Analytics**: Process analytics, completion rates, bottleneck identification, Nintex Hawkeye
- **Strengths**: SharePoint integration, document-centric, enterprise compliance
- **Gaps**: Expensive, complex setup, analytics are add-on, no submission-level drill-down
- **Pricing**: $25K+/year enterprise

## 4. Monday.com
- **Key Features**: Visual boards, kanban, timeline, Gantt, automations, dashboards
- **Approval Tracking**: Custom approval columns, status tracking, notifications, automations
- **Analytics**: Dashboard widgets, charts, pivot tables, time tracking
- **Strengths**: Beautiful UI, highly flexible, great collaboration, many integrations
- **Gaps**: Not purpose-built for approvals, no SLA tracking out-of-box, limited audit trail
- **Pricing**: $10-20/user/mo

## 5. Asana
- **Key Features**: Task management, workflows, timelines, portfolios, goals, forms
- **Approval Tracking**: Approval tasks, custom fields for status, rules-based routing
- **Analytics**: Dashboards, project status, workload management, portfolio insights
- **Strengths**: Great UX, powerful rules engine, portfolio-level visibility
- **Gaps**: Not designed for multi-level government approvals, no form-submission tracking, limited SLA
- **Pricing**: $10.99-24.99/user/mo

## 6. Jira Service Management
- **Key Features**: ITSM, approval workflows, SLA management, queues, knowledge base, automation
- **Approval Tracking**: Multi-approver, auto-approval rules, SLA timers, escalation policies
- **Analytics**: Custom dashboards, SLA reports, CSAT, queue analytics, JQL queries
- **Strengths**: Powerful SLA engine, deep customization, Confluence integration, audit logs
- **Gaps**: ITSM-focused (not general approvals), complex configuration, UI can overwhelm
- **Pricing**: $20-45/agent/mo

## 7. ServiceNow
- **Key Features**: Enterprise ITSM/ITOM, workflow engine, AI-powered automation, portal
- **Approval Tracking**: Enterprise approval engine, parallel/sequential, delegation, auto-escalation
- **Analytics**: Performance Analytics, predictive intelligence, real-time dashboards
- **Strengths**: Gold standard for enterprise, ML-powered predictions, comprehensive audit trail
- **Gaps**: Very expensive, 6-12 month implementation, overkill for form approvals
- **Pricing**: $100+/user/mo, $500K+ implementations

## 8. Power Automate + Power BI (Microsoft)
- **Key Features**: Flow automation, approval actions, Power BI dashboards, Teams integration
- **Approval Tracking**: Built-in approval connector, multi-level, custom responses, mobile
- **Analytics**: Power BI dashboards — any visualization possible, DAX queries, AI insights
- **Strengths**: Microsoft ecosystem, unlimited dashboard flexibility, AI-powered insights
- **Gaps**: Requires two products, Power BI learning curve, no unified approval dashboard
- **Pricing**: $15/user/mo (Automate) + $10/user/mo (Power BI)

## 9. Retool
- **Key Features**: Internal tool builder, drag-and-drop UI, database connectors, custom logic
- **Approval Tracking**: Custom-built — whatever you code. Tables, buttons, workflows
- **Analytics**: Custom charts, SQL queries, JavaScript transforms
- **Strengths**: Ultimate flexibility, can connect any API, fast to prototype
- **Gaps**: Not a product — it's a builder. No pre-built approval features, requires developers
- **Pricing**: $10-50/user/mo

## 10. Budibase
- **Key Features**: Low-code platform, auto-generated CRUD, custom workflows, self-hosted option
- **Approval Tracking**: Custom forms + automation, role-based access, status tracking
- **Analytics**: Built-in charts, custom queries, external DB connections
- **Strengths**: Open source option, self-hosted, fast to build, good for internal tools
- **Gaps**: Limited workflow engine, no native SLA, basic analytics, small ecosystem
- **Pricing**: Free (self-hosted), $50+/mo cloud

## 11. Appian
- **Key Features**: Low-code BPM, case management, AI/ML, RPA, process mining
- **Approval Tracking**: Enterprise case management, dynamic routing, parallel approvals, SLA
- **Analytics**: Process mining, real-time analytics, AI-driven insights, custom reports
- **Strengths**: Enterprise BPM leader, case management, process mining, AI features
- **Gaps**: Very expensive, long implementation, complex for simple approval flows
- **Pricing**: $75+/user/mo, $200K+ implementations

## 12. Pipefy
- **Key Features**: Process management, kanban boards, automations, forms, integrations
- **Approval Tracking**: Pipe phases as approval stages, card-based tracking, automations
- **Analytics**: Process metrics, SLA tracking, bottleneck identification, custom reports
- **Strengths**: Visual kanban approach, good mid-market fit, clean UI, affordable
- **Gaps**: Limited to kanban paradigm, no deep audit trail, basic predictive analytics
- **Pricing**: $25-30/user/mo

## 13. Zoho Creator
- **Key Features**: Low-code app builder, workflows, forms, reports, Zoho ecosystem
- **Approval Tracking**: Built-in approval tasks, multi-level, conditional, mobile approvals
- **Analytics**: Reports, dashboards, pivot tables, custom deluge scripting
- **Strengths**: Affordable, Zoho ecosystem, customizable, good for SMB
- **Gaps**: Limited enterprise features, basic UI, no real-time collaboration on approvals
- **Pricing**: $8-25/user/mo

## 14. JotForm (Native Tables/Reports)
- **Key Features**: Form builder, Tables (spreadsheet view), Report Builder, Approvals add-on
- **Approval Tracking**: Single-level approve/deny per submission, basic approval flow
- **Analytics**: Report Builder — charts, widgets, basic summary stats
- **Strengths**: Easy to use, good form builder, visual report builder
- **Gaps**:
  - ❌ No multi-level approval chain tracking (only single approve/deny)
  - ❌ No cross-form dashboard (each form is siloed)
  - ❌ No SLA tracking or escalation
  - ❌ No bottleneck analysis
  - ❌ No approver performance metrics
  - ❌ No kanban board for approvals
  - ❌ No audit trail beyond basic submission history
  - ❌ No predictive analytics
  - ❌ No delegation/out-of-office
  - ❌ No bulk actions across forms
  - ❌ No department-level scorecards
  - ❌ No calendar heatmap visualization
  - ❌ Limited export capabilities
- **Pricing**: $34-99/mo (form builder + approvals)
